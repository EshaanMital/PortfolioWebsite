Project 2: Rock Paper Scissors using Hand Gestures

Rock Paper Scissors using hand gestures is an exciting project that I developed for my portfolio website, utilizing Python and its libraries for computer vision. By leveraging the power of computer vision techniques, I created a program that enables users to play the classic game of Rock Paper Scissors using hand gestures.Using a webcam or any other suitable camera, the program captures real-time video input and analyzes the hand gestures of the player. The computer vision algorithms detect and recognize the hand gestures accurately, allowing the player to choose between rock, paper, or scissors. The program then competes against the player by randomly selecting its own gesture.To implement this project, I utilized Python's computer vision libraries, such as OpenCV, to process the video input and perform hand gesture recognition. By employing image processing techniques, I extracted relevant features from the video frames and classified them into different gestures using machine learning or rule-based algorithms.Through this project, I aimed to showcase my skills in computer vision, image processing, and Python programming. By combining these technologies, I was able to create an engaging and interactive game that brings a digital version of Rock Paper Scissors to life.
